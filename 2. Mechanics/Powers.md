Powers are conceptual in nature. They have more power over the world than first seems logical and the longer they are manifested the more obvious this becomes. 

They can be devided into some common categories
- [[Brick Power]]: Powers that allow for supernatural strength or durability. They are built like brick houses, strong and tough.

- [[Effector]]: Powers that allow the user to effect the enviroment around them to a greater degree.

- [[Genius]]: Powers that effect the inteligence or senses of the user to some degree. Allowing for displays of supernatural skill.

- [[Manipulator]]: Powers that manipulate the minds or actions of creatures near them. This could be people, animals or some constructs

- [[Mutant]]: Powers that visibly change the users anatomy to some degree. 

- [[Ranger]]: A class of powers that gives the user an new set of skills. Some of which may be completely impossible.

- [[Techie]]: Powers that allow the user to create different forms of super tech.
